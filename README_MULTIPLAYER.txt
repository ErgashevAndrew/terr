ФАЙЛЫ ИЗ ЭТОГО АРХИВА НУЖНО РАСКИНУТЬ ТАК:

terr/
  index.html                <- заменить
  server.js                 <- новый файл
  js/
    state.js                <- заменить
    menu.js                 <- заменить
    game.js                 <- заменить
    main.js                 <- заменить
    network.js              <- новый файл

ОСТАЛЬНЫЕ ТВОИ ФАЙЛЫ НЕ ТРОГАЕМ.

КАК ИГРАТЬ:

1. У хоста в папке terr открыть терминал.
2. Запустить сервер командой:
   node server.js

3. После этого у хоста открыть игру через Live Server.
4. В игре зайти в Online Mode.
5. Ввести ник.
6. Нажать Host.

ВТОРОЙ ИГРОК:

1. Должен быть в той же Wi‑Fi сети.
2. Должен узнать локальный IP хоста.
   На компьютере хоста в cmd команда:
   ipconfig
   Нужна строка IPv4 Address, например 192.168.0.15

3. Второй игрок открывает игру у себя.
4. Заходит в Online Mode.
5. Нажимает Find.
6. Вводит адрес в формате:
   192.168.0.15:3000
7. Нажимает Add.
8. Выбирает сервер в списке.
9. Нажимает Join.

ЧТО СЕЙЧАС УЖЕ РАБОТАЕТ:
- подключение к одному серверу
- один общий мир с сервера
- два игрока видят друг друга
- движение синхронизируется
- никнеймы видны над игроками

ЧТО ПОКА НЕ СДЕЛАНО:
- автоматический поиск серверов по сети
- общий ломаемый мир
- синхронизация установки/ломания блоков
- сервер не запускается кнопкой Host сам по себе, его пока надо запускать вручную
